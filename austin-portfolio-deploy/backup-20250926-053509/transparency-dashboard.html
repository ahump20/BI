Measurement Methodologies & Definitions

Data Points Calculation

  • Data points include individual player statistics, game events, and performance metrics
  • Counted from verified sources: MLB API, ESPN API, SportsRadar feeds
  • Excludes duplicate entries and invalid data points
  • Updated every 15 minutes during active game periods
  • Verified through automated data validation checks

Uptime Measurement

  • Monitored by third-party service (UptimeRobot) with 1-minute intervals
  • Uptime = (Total monitoring time - Downtime) / Total monitoring time
  • Planned maintenance windows excluded from calculations
  • Multiple global monitoring locations (US East, US West, EU)
  • HTTP 200 response required for "up" status

Response Time Measurement

  • Measured from initial HTTP request to complete response
  • 95th percentile metric excludes outliers and edge cases
  • Includes database query time, processing time, and network latency
  • Measured from 5 geographic locations across US
  • Excludes cold starts and first-time cache misses
